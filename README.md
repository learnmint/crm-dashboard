# CRM Dashboard

Full Sales CRM built with:
- Google Sheets (Database)
- Google Apps Script (Backend API)
- GitHub Pages (Frontend)

## Features
- Role-based login
- Leads management
- Payments with Razorpay
- Follow-ups + reminders
- Reports & analytics
- WhatsApp automation

## Deployment
1. Deploy Apps Script as Web App
2. Paste API URL in app.js
3. Enable GitHub Pages
4. Login using Users sheet

## Roles
ADMIN / MANAGER / TL / BDA
